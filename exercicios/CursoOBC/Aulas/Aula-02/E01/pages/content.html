Conceitos básicos JS

Tipos de dados primitivos

String

Valor textual que funciona como uma lista de caracteres.

Você tem 3 formas de escrever texto em JS:

  • Usando ""
  • Usando ''
  • Usando ``

Para abrir uma string você deve utilizar os []

Boolean

Valor lógico que pode ser verdadeiro (true) ou falso (false)

Number

Valor numérico que pode ser inteiro ou ponto flutuante (fracionário).

BigInt

Valor exclusivamente inteiro que permite ultrapassar o limite de "number"

Para chamar um BigInt usa-se:

  • 99n (sendo 99 um exemplo de número sucedido da letra n)

Null

Tipo usado para representar um valor nulo ou vazio

Undefined

Tipo usado para representar uma variável sem valor (não confundir com valor nulo)

Variáveis

Expressões Lógicas e Estrutura de Controle

E (and)

Representado por &&, num if depende das duas ou mais condições serem verdadeiras para a transmissão do True.

Ou (or) ||

Depende de apenas umas da condições para transmitir True.

Não (NOT) !

Transmite o oposto do que seria transmitido através das outras condições.

Estruturas condicionais

O que são:

Um recurso de linguagem que permite executar um bloco de código apenas se uma determinada expressão for verdadeira.

If / else

Usado para casos onde temos uma condição simples (se/senão). Pode-se adicionar else if em sequência para várias condições e atribuiições a elas.

Switch / case

Usado para casos onde queremos várias condições possíveis já conhecidas. Põe-se o valor dentro do switch (podendo ser uma variável perguntada anteriormente) e compará-la aos "cases" dentro do switch e se der match com algum deles o seu comando será executado. Porém, ele ao encontrar uma verdade ele mostrará todas as ações posteriores àquela que ele encontrou como verdadeira, para que isso não aconteça você deve utilizar a tag 'break'. O caso padrão para caso ele não encontre nenhuma verdade nas condições anteriores é o 'default'. O default pode ser utilizado também mesmo após ele encontrar uma verdade, desde que esteja no final e ali ele pode ter mais de uma função.

Estruturas de Repetição

O que são:

Um recurso de linguagem que permite executar um mesmo bloco de código várias vezes até que uma condição seja satisfeita.

While e do/while

Repete a execução enquanto a condição for verdadeira.

For

Repete a execução para um conjunto de condições e regras.

Funções

O que são:

Um recurso da linguagem que permite memorizar um bloco de código e executá-lo programaticamente sempre que quisermos. Fazemos isso através da definição de um nome para esse bloco, como o alert() e o prompt() do seu navegador.

Parâmetros de funções

São uma forma de informar valores para que as funções possam ter um comportameno dinâmico, ou seja, que gera um resultado diferente de acordo com os valores informados na execução.

Retorno de funções

É uma forma de devolver um resultado na execução da função. Permitindo que esse valor seja usado posteriormente fora do contexto da função.

Módulos

O que são:

Um mecanismo que permite compartilhar nosso código entre diferentes arquivos e projetos com mais facilidade, permitindo reaproveitar o nosso código (e o de outros devs).

CommonJS

Formato usado antes de 2015 quando a linguagem JS ainda não suportava módulos nativamente. São utilizados até hoje com a ajuda de ferramentas auxiliares.

Módulos ESM ou ECMAScript Modules

Recurso nativo de módulo da linguagem JS. Levou um tempo até ser adotado pelos navegadores, mas hoje já é a forma mais recomendada de se trabalhar com módulos na linguagem.

NPM (Node Package Manager)

O que é:

Um gerenciador de pacotes que também é um repositório de pacotes escritos em JavaScript para uso com Node.js. Ele ajuda no gerenciamento geral de módulos extras em um projeto JavaScript. Uma biblioteca.

Empacotadores

O que são:

Ferramentas que empacotam o código JavaScript necessário para o funcionamento de um projeto em uma saída otimizada e fácil de incluir em qualquer site, removendo a necessidade do npm e da pasta "node_modules".

Estruturas de dados

O que são:

São formas mais complexas e organizadas de armazenar e gerenciar dados em um programa de computador. Elas permitem o armazenamento, acesso e manipulação eficiente de informações, como números, textos e objetos, de maneira que atenda às necessidades específicas de processamento.

Objetos

Um tipo de dado mais complexo que permite armazenar vários valores em uma única variável. Funciona como um dicionário, onde definimos propriedades através de pares chave-valor.

Arrays

O que são:

Assim como os objetos são um tipo de dado mais elaborado que permite armazenar vário valores em uma única variável. Os arrays (ou vetores) identificam os elementos pela sua posição ao invés de uma chave, ou seja, funcionam como uma lista ordenada de valores.

Métodos de Array

  • Push: adiciona elementos no fim do array
  • Pop: remove o último elemento do array
  • Shift: remove o primeiro elemento do array
  • Unshift: adiciona elementos no começo do array
  • Concat: combina dois ou mais arrays em um
  • Join: junta os elementos de uma string
  • slice: retorna uma cópia de um pedaço do array
  • splice: altera o conteúdo de um array, adicionando, removendo ou subtituindo elementos
