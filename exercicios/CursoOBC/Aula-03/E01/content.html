Conceitos básicos JS

Tipos de dados primitivos

String

Valor textual que funciona como uma lista de caracteres.

Você tem 3 formas de escrever texto em JS:

  • Usando ""
  • Usando ''
  • Usando ``

Para abrir uma string você deve utilizar os []

Boolean

Valor lógico que pode ser verdadeiro (true) ou falso (false)

Number

Valor numérico que pode ser inteiro ou ponto flutuante (fracionário).

BigInt

Valor exclusivamente inteiro que permite ultrapassar o limite de "number"

Para chamar um BigInt usa-se:

  • 99n (sendo 99 um exemplo de número sucedido da letra n)

Null

Tipo usado para representar um valor nulo ou vazio

Undefined

Tipo usado para representar uma variável sem valor (não confundir com valor nulo)

Variáveis

O que são variáveis?

Uma forma de guardar na memória valores para usá-los posteriormente no programa

Var, let ou const?

Const tem valores imutáveis

Let it change, ou seja, o 'Let' é mutável e substitui bem o var
